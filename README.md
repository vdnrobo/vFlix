# vFlix

vFlix (vdn + flight + X) is a fork of the open-source drone project [Flix](https://github.com/okalachev/flix) (authored by Oleg Kalachev). It is based on ESP32, programmed in Arduino IDE, CAD - KOMPAS-3D v23.
